# DECISIONS — журнал ключевых решений

> Ведёт агент, дополняет PO. Только решения с долгосрочными последствиями:
> выбор из альтернатив, который дорого переиграть. Не журнал изменений!
> Новые записи — сверху. Читается точечно (поиском по теме), не целиком.

Формат записи:

---

## D-001: <краткая суть решения> — <дата>

**Контекст:** <какая проблема/развилка возникла, в рамках какой стори.>

**Решение:** <что выбрано.>

**Альтернативы:** <что отвергнуто и почему, 1–2 строки.>

**Последствия:** <что это значит для будущих модулей/стори.>
