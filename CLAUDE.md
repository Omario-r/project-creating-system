# CLAUDE.md — маршрутизатор ролей

Правила системы — в `system/`. Адаптации этого проекта — в `PROJECT.md`.
В любой роли читай оверлей `PROJECT.md` (стек, каталог кода, линтер) вместе
с правилами выбранной роли.

Роль — по команде PO:

- «консультация» / обсудить идею, Vision, эпик → `system/ADVISOR.md`
  (Советник: спарринг по идее/Vision/эпику, черновики).
- «декомпозируй эпик <X>» / «нарежь стори <N> на задачи» →
  `system/POSTANOVSHCHIK.md` (Постановщик: эпик→стори и стори→ТЗ задач).
- «возьми/выполни задачу», «продолжи стори со следующей задачи» →
  `system/AGENT.md` (Исполнитель: одно ТЗ → код).

Сомневаешься, какая роль нужна, — спроси PO, не угадывай.
